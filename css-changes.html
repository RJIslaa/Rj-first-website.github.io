CSS Changes

This page explains the CSS customizations I made to make the website look pretty.

Button Styling

I customized the Bootstrap button classes to give them unique colors:

  • Primary buttons (like the Interests button): Green background (#4caf50) that darkens on hover (#45a049).
  • Secondary buttons (like this page's button): Blue background (#2196F3) that darkens on hover (#1976D2).

These colors make the buttons stand out and provide a nice visual feedback when hovering.

Game Cell Layout

For the games table on the Interests page, I added a custom .game-cell class:

  • Uses flexbox to align images and text side by side.
  • Images are sized to 80x80 pixels with rounded corners and object-fit cover for consistent display.
  • Adds a small gap between the image and the game name for better spacing.

This makes the table more visually appealing and easier to read.
