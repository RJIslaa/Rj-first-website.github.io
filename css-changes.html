CSS Changes

This page explains the CSS customizations I made to make the website look pretty.

Button Styling

I customized the Bootstrap button classes to give them unique colors:

  • Primary buttons (like the Interests button): Green background (#4caf50) that darkens on hover (#45a049).
  • Secondary buttons (like the CSS Changes button): Blue background (#2196F3) that darkens on hover (#1976D2).
  • Purple buttons (like the Resume button): Purple background (#9c27b0) that darkens on hover (#7b1fa2).

These colors make the buttons stand out and provide a nice visual feedback when hovering.

Social Media Icons

I added Font Awesome icons for social media links with custom styling:

  • Icons are displayed inline with 2.5rem font size and spaced evenly.
  • Default color is dark (#333) and changes to purple (#9c27b0) on hover.
  • Icons scale up 1.2x on hover with a smooth transition effect.

Centered Text

All headings (h2) and paragraphs (p) are now center-aligned for better visual balance and readability.

Game Cell Layout

For the games table on the Interests page, I added a custom .game-cell class:

  • Uses flexbox to align images and text side by side.
  • Images are sized to 80x80 pixels with rounded corners and object-fit cover for consistent display.
  • Adds a small gap between the image and the game name for better spacing.

This makes the table more visually appealing and easier to read.

Navbar system

I implemented a responsive navbar using Bootstrap's built-in classes.
